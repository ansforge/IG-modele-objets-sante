Obligations
ExerciceProfessionnel 0..* Element Informations décrivant notamment la profession exercée, l'identité d'exercice d'un professionnel et le cadre de son exercice (civil, agent public, etc.).Il peut exister plusieurs exercices professionnels pour une personne à un instant donné.
civiliteExercice 0..1 Coding Civilité d’exercice du professionnel.
Binding: ValueSet[https://interop.esante.gouv.fr/ig/fhir/mos/ValueSet/CiviliteExercice-vs|0.1.0] (preferred)
nomExercice 0..1 string Nom sous lequel exerce le professionnel.
prenomExercice 0..1 string Prénom sous lequel exerce le professionnel.
profession 0..1 Coding Profession exercée ou future profession de l'étudiant.
Binding: ValueSet[https://interop.esante.gouv.fr/ig/fhir/mos/ValueSet/profession-vs|0.1.0] (preferred)
typeProfession 0..1 Coding Indique le type de profession de la personne.Exemples :** Professionnel de santé** Usager de titre** Professionnel secteur social** etc.
Binding: ValueSet[https://interop.esante.gouv.fr/ig/fhir/mos/ValueSet/typeProfession-vs|0.1.0] (preferred)
categorieProfessionnelle 0..1 Coding Indique si le professionnel exerce sa profession en tant que :** M: Agent public** C: Civil** F: Fonctionnaire ** E: EtudiantL'autorité d'enregistrement du professionnel dépend de la catégorie de sa profession.
Binding: ValueSet[https://interop.esante.gouv.fr/ig/fhir/mos/ValueSet/CategorieProfessionnelle-vs|0.1.0] (preferred)
dateEffetExercice 0..1 date Date à partir de laquelle le professionnel exerce cette profession.
dateFinEffetExercice 0..1 date Date à partir de laquelle le professionnel n’exerce plus cette profession.
departementPremierReferencementAE 0..1 Coding Code du département du premier référencement, pour un référencement de nature « inscription à l’ordre ».
Binding: ValueSet[https://interop.esante.gouv.fr/ig/fhir/mos/ValueSet/DepartementOM-vs|0.1.0] (preferred)
boiteLettresMSS 0..* ​https://interop.esante.gouv.fr/ig/fhir/mos/StructureDefinition/BoiteLettreMSS Boîte(s) aux lettres du service de messagerie sécurisée de santé (MSS) rattachée(s) à l'exercice professionnel du professionnel.
